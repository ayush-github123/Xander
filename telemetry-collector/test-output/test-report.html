Troubleshooting

If metrics.db is empty or missing:

  • Check collector logs: kubectl logs -n telemetry-system -l app=telemetry-collector
  • Verify pod discovery: kubectl exec -it <collector-pod> -n telemetry-system -- ls -la /app/
  • Check cgroup access: kubectl describe pod -n telemetry-system <collector-pod>

If no data for test pods:

  • Verify test pods are running: kubectl get pods -n test-scenarios
  • Check pod labels: kubectl get pods -n test-scenarios --show-labels
  • Verify collector can see pods: kubectl exec <collector-pod> -n telemetry-system -- curl -k https://127.0.0.1:10250/pods
